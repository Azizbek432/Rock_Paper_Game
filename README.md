# 🪨 📄 ✂️ Rock-Paper-Scissors Game

Ushbu loyiha klassik **"Tosh-qog'oz-qaychi"** o'yinining veb-talqinidir. Loyiha mashhur **Bro Code** YouTube kanali darslari asosida yaratilgan bo'lib, unda asosiy e'tibor JavaScript-dagi shart operatorlari va DOM bilan ishlashga qaratilgan.

---

## 🇺🇿 Loyiha haqida
O'yin foydalanuvchi va kompyuter o'rtasida bo'lib o'tadi. Har bir tanlovdan so'ng g'olib aniqlanadi va hisob (score) yangilanib boradi. Bu loyiha orqali men JavaScript-da tasodifiy sonlar bilan ishlash (`Math.random()`) va foydalanuvchi harakatlarini kuzatishni o'rgandim.

### ✨ Xususiyatlari:
* **Interactive UI:** Tugmalar orqali oson boshqariladigan interfeys.
* **Real-time Logic:** G'olibni darhol aniqlash va natijani ko'rsatish.
* **Score Tracker:** O'yin davomida natijalarni hisoblab borish.
* **Clean Design:** Bro Code uslubidagi sodda va tushunarli vizual ko'rinish.

---

## 🇬🇧 About the Project
This is a web-based version of the classic **"Rock-Paper-Scissors"** game, inspired by **Bro Code's** tutorials. The project focuses on JavaScript logic, including conditional statements, function handling, and DOM manipulation.

### ✨ Key Features:
* **User vs Computer:** Play against a computer that makes randomized choices.
* **Dynamic Results:** Instant feedback on who won each round.
* **Score Management:** Keeps track of the player's and computer's scores.
* **Responsive Styling:** A clean layout built with HTML and CSS.

---

## 🛠 Texnologiyalar | Technologies
* **HTML5** — O'yin elementlari va strukturasi uchun.
* **CSS3** — Tugmalar dizayni va layout uchun.
* **JavaScript** — O'yin mantig'i, `Math.random()` va hodisalarni boshqarish uchun.

## 🎮 Qanday o'ynash kerak? | How to Play?
1. Repozitoriyani yuklab oling.
2. `index.html` faylini brauzerda oching.
3. Tosh, qog'oz yoki qaychi tugmalaridan birini bosing.
4. Natijani ko'ring va kompyuterni yengishga harakat qiling!
